# Delhi '26 — BWF World Championships concept site

A single-page concept/demo site for the 2026 BWF World Championships, being hosted in New Delhi (Aug 17–23, 2026) — India's first time hosting since 2009.

Built as a design and frontend concept: live countdown, India's medal legacy timeline, a filterable Team India roster, day-by-day schedule, venue overview, and ticket tier cards.

**Live demo:** _add your Vercel URL here after deploying_

## Stack

Plain HTML, CSS, and vanilla JavaScript — no framework, no build step, no dependencies. Fonts loaded from Google Fonts (Fraunces, Space Grotesk, JetBrains Mono).

## Run locally

Just open `index.html` in a browser. No install needed.

## Deploy

This is a static site, so it deploys as-is on Vercel, Netlify, or GitHub Pages with zero configuration.
